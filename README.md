## Clone the project

```
$ git clone https://github.com/foruscommunity/go-rest-api-example
$ cd go-rest-api-example
```

---

## Execute

First copy the `dev/.env.example` to the `dev/.env`

Then run the application:

```sh
go run main.go
```
